CSS BOX MODEL

CSS box model is basically a box that wraps around HTML elemrnts

Properties Of Box Model

Margin - It sets an area/margin on all the four sides of an element.pushing the elements away outside the box.Its transparent

Padding - Clears an area around the element. Its transparent

Content - Its the content of the page,which includes the text,images and also the videos.

Border - This is a border that goes around the padding and the content of the page


How to calculate the total dimension of a box with width:400px; padding:20px; margin:20px; border:2px solid #EEE;

To get the total dimension you add the width,To calculate the full size of an element, padding, borders and margins.

Solution

width + padding + marggin + border

400 + 20 + 20 + 2 = 442

Total dimension = 442px
